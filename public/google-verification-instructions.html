Google Search Console Verification

This is a template file. Follow the instructions below:

Step 1: Get Your Verification File

  1. Go to Google Search Console
  2. Add property: https://param20h.me
  3. Choose "HTML file upload" verification method
  4. Download the verification file (e.g., google1234567890abcdef.html)

Step 2: Add File to Your Project

  1. Copy the downloaded file (google[xxxxx].html)
  2. Paste it into: d:\param20h.github.io\public\
  3. The file will be accessible at: https://param20h.me/google[xxxxx].html

Step 3: Deploy and Verify

  1. Commit and push your changes to GitHub
  2. Wait for GitHub Actions deployment to complete (2-3 minutes)
  3. Go back to Google Search Console
  4. Click "Verify" button

Alternative: Meta Tag Method

}

Steps:

  1. Get the verification code from Google Search Console
  2. Replace "google-site-verification-code" in app/layout.tsx
